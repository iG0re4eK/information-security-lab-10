Алгоритм исчисления порядка

n = p - 1

Алгоритм:

Шаг 1. Выбирается факторная база S = {p1, p2, p3, ..., pt}. (Если G = Zp*, то S состоит из t первых простых чисел.)

Шаг 2. Выбрать случайное k: 0 ≤ k < n и вычислить gk.

Шаг 3. Попытаться разложить gk по факторной базе:
gk = ∏i=1t piai, ai ≥ 0.
Если это не удалось, вернуться на Шаг 2.

Шаг 4. Логарифмируя обе части получившегося выражения, получаем
k ≡ ∑i=1t ai logg(pi) (mod n) *
В этом выражении неизвестными являются логарифмы.
Это сравнение с t неизвестными следует запомнить.

Шаг 5. Если сравнений вида (*), полученных на Шаге 4, меньше, чем t + c, то вернуться на Шаг 2.

Шаг 6. Решить систему t + c сравнений с t неизвестными вида (*), составленную на Шагах 2-5.

Шаг 7. Выбрать случайное k: 0 ≤ k < n и вычислить agk.

Шаг 8. Попытаться разложить agk по факторной базе:
agk = ∏i=1t pibi, bi ≥ 0.
Если это не удалось, вернуться на Шаг 7.

Шаг 9. Логарифмируя обе части последнего равенства, получаем
x = logga = ∑i=1t bi logg(pi) - k (mod n),
где logg(pi) (1 ≤ i ≤ t) вычисленны на Шаге 6 как решение системы сравнений.

Выход:
x = logg(a) mod n.
